# Jivora Core

## Overview
A Node.js web application using Express, serving static files from the `public/` directory.

## Project Architecture
- **Runtime**: Node.js 20
- **Framework**: Express 5
- **Entry point**: `index.js`
- **Static assets**: `public/`
- **Port**: 5000 (bound to 0.0.0.0)

## Getting Started
Run `npm start` to start the server on port 5000.
